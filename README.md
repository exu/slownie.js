Slownie.js
==========

Little library for converting numbers into polish words. 
Ported from C version at found at http://4programmers.net/Pastebin/1668

Installation
------------

```
$ npm install slownie
```

Usage
-----
```

var Slownie = new require('slownie').Slownie();
```
